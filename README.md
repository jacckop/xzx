# Whitegram Languages v4

ديلب مستقل لتعريب وترجمة قسم **Whitegram Features** في Whitegram 12.8 (55)، مع دائرة لغة عائمة قابلة للتحريك وقائمة احترافية لاختيار اللغة فورياً.

لا يعتمد المشروع على Substrate أو ElleKit، وهو مخصص للحقن المباشر داخل IPA ثم إعادة توقيع التطبيق.

## اللغات

اللغة الافتراضية هي العربية، والخيارات المتاحة:

- العربية
- English
- Español
- فارسی
- Français
- Português

الإنجليزية تعيد النصوص الأصلية، أما اللغات الخمس الأخرى فتحتوي كل واحدة منها على **742 ترجمة** مطابقة لمفاتيح قسم Whitegram.

## دائرة اختيار اللغة

- تظهر كدائرة صغيرة عائمة تحمل رمز الكرة الأرضية.
- يمكن سحبها إلى أي مكان، ثم تلتصق تلقائياً بأقرب جانب.
- يُحفظ موقعها بعد إغلاق التطبيق.
- عند الضغط عليها تظهر قائمة زجاجية متحركة تضم اللغات الست.
- تظهر علامة اختيار بجانب اللغة الحالية.
- يتغير اتجاه الواجهة تلقائياً بين RTL للعربية والفارسية وLTR لبقية اللغات.
- يتم تطبيق اللغة فوراً من دون الحاجة إلى إعادة تشغيل Whitegram.

أسفل القائمة توجد الحقوق:

`@iKiraPlus`

مع شعار Telegram، والضغط عليها يفتح الحساب مباشرة عبر `tg://resolve?domain=iKiraPlus` أو رابط الويب كخيار احتياطي.

## نظام الترجمة

يستمر المشروع باستخدام إصلاح NodeFix الآمن:

- اعتراضات UIKit المحدودة والآمنة.
- تعريب عُقد Telegram المبنية على Texture / AsyncDisplayKit.
- دعم `ImmediateTextNode` و`ImmediateTextView` و`TextNodeWithEntities`.
- دعم واجهات `ComponentFlow.Text`.
- إعادة فحص الصفحات المرئية عند تغيير اللغة.
- إمكانية التحويل من أي لغة مدعومة إلى أي لغة أخرى، وليس من الإنجليزية فقط، عبر إعادة النص إلى مفتاحه الأصلي ثم تطبيق اللغة الجديدة.
- عدم اعتراض مُنشئات تسجيل الدخول أو جميع كلاسات التطبيق، لتجنب كرش تسجيل الدخول الذي حدث في الإصدار الأول.

## البناء عبر GitHub Actions

1. ارفع محتويات هذا المجلد إلى جذر مستودع GitHub.
2. افتح تبويب **Actions**.
3. اختر **Build Whitegram Languages Dylib**.
4. اضغط **Run workflow**.
5. نزّل Artifact باسم `WhitegramLanguages-v4`.

سيحتوي على:

- `WhitegramArabic.dylib`
- `SHA256.txt`
- `coverage-report.txt`

تم الإبقاء على اسم الديلب `WhitegramArabic.dylib` حتى يمكنه استبدال إصدار NodeFix السابق مباشرة داخل مشروع الحقن.

## الحقن الصحيح

استخدم IPA أصلياً ونظيفاً، أو احذف إصدار الديلب السابق بالكامل قبل حقن v4.

ضع الديلب داخل:

`Payload/Telegram.app/Frameworks/WhitegramArabic.dylib`

ثم أضف تحميله إلى Mach-O الرئيسي باسم:

`@rpath/WhitegramArabic.dylib`

بعدها أعد توقيع التطبيق بالكامل. يجب ألا توجد إحالتان للديلب نفسه داخل `LC_LOAD_DYLIB`.
