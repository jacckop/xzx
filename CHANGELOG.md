# Changelog

## 4.0.0 — Languages Overlay

- إضافة دائرة لغة عائمة قابلة للسحب مع حفظ موقعها.
- إضافة قائمة زجاجية متحركة لاختيار 6 لغات.
- جعل العربية اللغة الافتراضية.
- دعم الإنجليزية والإسبانية والفارسية والفرنسية والبرتغالية.
- إضافة 742 ترجمة كاملة لكل لغة غير الإنجليزية.
- دعم التحويل المباشر بين اللغات وإعادة النصوص الإنجليزية الأصلية.
- تطبيق RTL تلقائياً للعربية والفارسية وLTR لبقية اللغات.
- إضافة حقوق `@iKiraPlus` مع شعار Telegram ورابط مباشر للحساب.
- الإبقاء على NodeFix الآمن الذي يترجم عُقد Texture/AsyncDisplayKit من دون المساس بتسجيل الدخول.

## 3.0.0 — NodeFix

- إصلاح بقاء عناصر إعدادات Whitegram باللغة الإنجليزية.
- إضافة مترجم مرئي لعُقد AsyncDisplayKit/Texture.
- عدم إعادة الاعتراض الواسع الذي تسبب بكرش تسجيل الدخول في الإصدار الأول.
